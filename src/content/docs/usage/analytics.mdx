## Stats

Show overall archive statistics:

```bash
msgvault stats
```

Displays total message count, account breakdown, date range, storage size, and attachment count.

<img src="/stats.svg" alt="msgvault stats command output" style="width: 100%; max-width: 800px; margin: 1rem 0; border-radius: 6px;" />

## List Senders

```bash
# Top 20 senders
msgvault list-senders --limit 20
```

## List Domains

```bash
# Top 20 sender domains
msgvault list-domains --limit 20
```

## List Labels

```bash
# All labels with message counts
msgvault list-labels
```

<img src="/list-senders.svg" alt="msgvault list-senders command output" style="width: 100%; max-width: 800px; margin: 1rem 0; border-radius: 6px;" />

These commands query the SQLite database directly. For interactive exploration with drill-down, use the [TUI](/usage/tui/).
